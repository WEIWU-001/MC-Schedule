## 🚀 快速上手指南

### 📋 准备工作

在开始之前，你需要准备：
1. **Python 3.8+**（没有的话赶紧去装！）
2. **一个邮箱账号**（用来发验证码和通知）
3. **一颗热爱MC的心** ❤️

### 🛠️ 安装步骤

```bash
# 1. 先把项目克隆下来
git clone https://github.com/你的用户名/MC-Schedule.git
cd MC-Schedule

# 2. 创建虚拟环境（好习惯要养成）
python -m venv venv

# 3. 激活虚拟环境
# Windows 用户：
venv\Scripts\activate
# Mac/Linux 用户：
source venv/bin/activate

# 4. 安装依赖（等着就行，泡杯茶）
pip install -r requirements.txt

# 5. 配置环境变量（重点！）
cp .env.example .env
# 打开 .env 文件，填写你的邮箱信息
```

### ⚙️ 配置 `.env` 文件

用记事本打开 `.env` 文件，填入以下内容：

```ini
# 服务器密钥（随便写一串字符，别让别人知道）
SECRET_KEY=这里写一串复杂的字符串比如a1b2c3d4e5f6

# 开发模式（上线后改成 false）
DEBUG=false

# 邮箱配置（以QQ邮箱为例）
MAIL_SERVER=smtp.qq.com
MAIL_PORT=587
MAIL_USE_TLS=true
MAIL_USERNAME=你的QQ邮箱@qq.com
MAIL_PASSWORD=你的邮箱授权码（不是密码！）

# 超级管理员ID（等注册后填）
SUPER_ADMIN_ID=1
```

### 🎉 启动服务

```bash
# 运行起来！
python app.py
```

然后打开浏览器访问：
- **前台首页**：http://localhost:5000
- **管理后台**：http://localhost:5000/admin

## 📝 使用流程详解

### 🔧 第一步：初始化配置

1. 启动服务后，先注册一个账号
2. 打开数据库工具（比如 DB Browser for SQLite），找到 `database.db`
3. 在 `users` 表中找到你注册的用户，记一下 `id`
4. 把这个 `id` 填到 `.env` 的 `SUPER_ADMIN_ID` 里
5. 重启服务，现在你就是超级管理员啦！

### 👤 第二步：用户注册

玩家访问首页后：
1. 点击右上角「注册」按钮
2. 输入邮箱，点击「获取验证码」（5分钟内有效）
3. 收到邮件后输入验证码
4. 设置用户名和密码，完成注册！

### 🌟 第三步：申请成为档主

玩家想创建档期？需要先申请档主：
1. 登录后点击「申请成为档主」
2. 填写档主介绍（比如擅长什么玩法）
3. 提交申请，等待管理员审核
4. 审核通过后就能创建档期啦！

### 📅 第四步：创建档期

档主登录后台：
1. 点击左侧「档期管理」
2. 点击「创建档期」按钮
3. 填写信息：
   - 档期标题（比如「周末生存挑战」）
   - 开始时间（选个大家都有空的时间）
   - 结束时间
   - 最大人数
   - 活动描述（写清楚玩法、规则）
4. 点击发布，玩家就能看到啦！

### 🎫 第五步：玩家预约

玩家看到档期后：
1. 点击档期卡片查看详情
2. 如果名额还有，点击「立即预约」
3. 预约成功！等待活动开始
