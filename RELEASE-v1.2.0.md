# MC-Schedule v1.2.0 发布说明

---

## 🆕 新增功能

| 功能 | 说明 |
|------|------|
| **用户设置页面** | 新增 `/settings` 页面，支持修改昵称、邮箱、密码、头像 |
| **网站运行时间** | 首页底部显示网站运行时间，可在后台设置起始时间 |
| **bcrypt密码加密** | 密码加密升级为bcrypt，比MD5更安全 |
| **超级管理员配置** | 通过 `.env` 配置 `SUPER_ADMIN_ID`，使用该昵称注册自动成为最高管理员 |

---

## 🔧 优化改进

| 优化项 | 说明 |
|-------|------|
| **管理后台加载** | 修复后台显示"加载中"问题，添加完善的错误处理 |
| **README文档** | 更新联系方式，添加QQ群号 |
| **界面样式** | 优化监控卡片样式、运行时间徽章样式 |

---

## 🗑️ 删除功能

| 删除项 | 说明 |
|-------|------|
| **腾讯云TMS审核** | 删除腾讯云内容审核，保留本地关键词过滤 |
| **网站公告** | 删除旧的网站公告功能 |
| **背景图片设置** | 临时隐藏，功能开发中 |
| **测试文件** | 删除 check_db.py、points.html 等测试文件 |

---

## 🔒 安全改进

- 密码加密从MD5升级为bcrypt
- 添加向后兼容：旧的MD5密码仍可登录
- 删除敏感信息泄露风险（腾讯云密钥）

---

## 📋 环境配置更新

`.env` 文件新增配置项：
```ini
# 最高管理员昵称（使用此昵称注册自动成为超级管理员）
SUPER_ADMIN_ID=惟五
```

## 🚀 升级方式

**方式一：使用后台一键更新**
进入管理后台 → 系统更新 → 点击检查更新

**方式二：手动更新**
1. 下载最新版本压缩包
2. 解压覆盖原文件（注意备份 `.env` 和 `database.db`）
3. 重启服务

---

**建议升级方式**：使用后台「🚀 系统更新」功能一键更新，或手动拉取最新代码后重启服务。
