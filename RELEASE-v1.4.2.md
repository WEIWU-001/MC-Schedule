# MC档期排期站 v1.4.2 更新日志

## 🎉 新增功能

### 1. 滚动公告与进站提醒
- 在主界面导航栏下方添加横向滚动公告栏
- 添加进站弹窗提醒功能，首次访问或公告更新时自动弹出
- 支持 24 小时内不再提醒（基于 localStorage）
- 公告内容可在后台「网站信息设置」中配置

### 2. 批量发送邮件功能
- 在后台「邮箱池管理」中添加批量发送邮件模块
- 支持选择发送邮箱（主邮箱/副邮箱）
- 可设置发送间隔（1秒/2秒/3秒/5秒/10秒）
- 实时显示发送进度和预计剩余时间
- 支持 Markdown 格式邮件内容
- 自动将 Markdown 转换为带样式的 HTML 邮件

## 🔧 优化改进

### 1. 邮件发送优化
- 邮件发送方法支持 HTML 格式
- 添加 Markdown 转 HTML 转换函数
- 添加邮件内容样式模板

### 2. 其他优化
- 修复 `btoa()` 函数无法处理中文的问题
- 添加 `get_next_email()` 方法用于邮箱轮换

## 📋 更新内容

| 文件 | 说明 |
|------|------|
| `VERSION` | 更新版本号至 v1.4.2 |
| `app.py` | 添加批量发送邮件 API、Markdown 转换、邮箱轮换方法 |
| `templates/admin.html` | 添加批量发送邮件界面和样式 |
| `static/js/app.js` | 添加滚动公告和进站提醒逻辑 |

## ⚠️ 注意事项

1. 邮件批量发送时建议使用 3 秒或更长间隔，避免被邮件服务商拦截
2. 进站提醒使用 localStorage 记录关闭状态，清除浏览器缓存后会重新显示
3. 如果安装了 `marked` 模块，Markdown 转换会使用更完善的解析器
