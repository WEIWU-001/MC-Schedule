# MC档期排期站 v1.4.2 更新日志

## 🎉 新增功能

### 1. 滚动公告系统
- 在主界面导航栏下方添加横向滚动公告栏，实时展示网站公告信息
- 支持渐变背景和无限滚动动画
- 鼠标悬停时暂停滚动，方便阅读
- 响应式设计，适配移动端显示

### 2. 进站提醒弹窗
- 网站加载完成后自动弹出公告提醒弹窗
- 支持 24 小时内不再提醒（基于 localStorage）
- 公告内容更新时自动重新显示
- 点击"知道了"按钮关闭弹窗
- 游客可见，无需登录即可查看

### 3. 批量发送邮件功能
- 在后台「邮箱池管理」中添加批量发送邮件模块
- 支持选择发送邮箱（主邮箱/副邮箱），可手动切换
- 可设置发送间隔（1秒/2秒/3秒/5秒/10秒），降低被拦截风险
- 实时显示发送进度条、成功/失败数量和预计剩余时间
- 支持随时停止发送
- 支持 Markdown 格式邮件内容，自动转换为带样式的 HTML
- 自动过滤无效邮箱地址

## 🔧 优化改进

### 1. 邮件系统优化
- 邮件发送方法支持 HTML 格式
- 添加 Markdown 转 HTML 转换函数
- 添加邮件内容样式模板，支持标题、粗体、斜体、列表、链接等格式
- 邮件底部自动添加发送时间和系统标识

### 2. 其他优化
- 修复 `btoa()` 函数无法处理中文的问题，改用 `encodeURIComponent()`
- 移除自动轮换功能，改为手动选择邮箱
- 下拉框显示邮箱类型和地址（如：主邮箱（***@example.com））

## ⚠️ 注意事项

1. 邮件批量发送时建议使用 3 秒或更长间隔，避免被邮件服务商拦截
2. 进站提醒使用 localStorage 记录关闭状态，清除浏览器缓存后会重新显示
3. 如果安装了 `marked` 模块，Markdown 转换会使用更完善的解析器
4. 公告内容为空时，滚动公告栏和弹窗会自动隐藏
5. 游客无需登录即可查看公告信息，适合发布通知给所有访问者
