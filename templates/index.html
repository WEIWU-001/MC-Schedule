📋 用户注册须知

欢迎注册 MC整合包档期排期站！

一、账号注册

1. 游戏ID需唯一且不可重复，注册成功后无法修改，请谨慎填写
2. 每位用户仅可注册一个账号，禁止注册多个账号

二、邮箱验证

1. 注册需绑定有效邮箱地址，用于接收验证码和重要通知
2. 请确保邮箱地址真实有效，验证码将通过邮件发送
3. 邮箱将用于密码找回等安全验证，请妥善保管

三、密码设置

1. 密码建议包含字母、数字和特殊字符，提高账号安全性
2. 请勿使用简单密码（如123456、生日等）
3. 请勿将账号密码透露给他人

四、预约功能

1. 预约档期后，开服前系统将发送邮件提醒
2. 请及时查看邮件，确保不错过预约档期的开服时间
3. 如需取消预约，可在开服前随时取消

五、行为规范（重要）

1. 禁止发表脏话、粗话或不文明言论
2. 禁止发布违规违法内容，包括但不限于：
    • 色情、暴力、血腥内容
    • 诈骗、钓鱼、木马等有害信息
    • 侵犯他人隐私或知识产权的内容
3. 禁止恶意刷屏、重复提交相同内容
4. 禁止模仿管理员或官方人员身份

六、违规处理

1. 违反以上规定的用户，将根据情节严重程度受到以下处罚：
    • 警告并删除违规内容
    • 禁言24小时至7天
    • 永久封禁账号
2. 管理员有权在不通知的情况下删除违规内容

七、账号安全

1. 请勿将账号借给他人使用，由此造成的后果由账号注册者承担
2. 如发现账号异常，请立即联系管理员
3. 请定期更换密码，确保账号安全

📋 成为档主须知

申请成为档主

一、身份验证

1. 申请需提供服务器IP或Q群号进行身份验证
2. 管理员将核实您是否为真实的服务器管理者
3. 每位用户仅可申请一个档主账号

二、审核流程

1. 提交申请后，管理员将进行人工审核
2. 审核时间一般为24-48小时
3. 审核结果将通过邮件通知，请保持邮箱畅通
4. 若审核拒绝，请联系管理员

三、档期发布规则（重要）

1. 每天最多发布3个档期，请合理安排
2. 每个档期每天只能修改1次，请在发布前仔细核对信息
3. 档期内容包括：日期、时间、服务器ID、联系方式等
4. 发布后需等待管理员审核通过才能显示

四、档主职责

1. 确保发布的档期信息真实有效
2. 按约定时间准时开服
3. 保持联系方式畅通，及时回复用户咨询
4. 妥善管理自己发布的档期内容

五、行为规范

1. 禁止发布虚假、欺诈性质的档期信息
2. 禁止冒用他人服务器信息
3. 禁止发布违法游戏内容
4. 禁止在评论区发表脏话、粗话等不文明言论
5. 禁止骚扰或攻击其他用户

六、违规处理

1. 违反规定将根据情节受到以下处罚：
    • 警告并要求整改
    • 删除违规档期
    • 暂停档主权限3-30天
    • 永久取消档主资格
2. 情节严重者将移交相关部门处理

七、免责声明

1. 因不可抗力导致无法开服，平台不承担责任
2. 用户间交易行为与平台无关
3. 档主需自行承担发布内容的法律责任
